# Simple To-Do List Manager

A simple Python program to manage your To-Do List.

## Features
- Add a new task
- View all tasks
- Mark a task as done
- Delete a task
- Exit the program

## How to Use
1. Download or clone this project
2. Run the `todolist.py` file using Python 3
3. Select menu options to manage your tasks

```bash
python todolist.py
